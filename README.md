Api simples feita com Node e Fastify para treinar e a criação de APIs de forma simples e local.
O sistema:
- Estrutura legível de pastas
- Rotas criadas para post e docs
- Middlewaresimple de autenticacção
- Criação e exclusão de posts
- Criação de comentários
- Toggle para dar like em um comentário
- Documentação feita com Swagger
